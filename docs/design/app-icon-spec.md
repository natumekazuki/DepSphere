# DepSphere アプリアイコン仕様

## 目的
- Windows アプリとしての識別性を高めるため、DepSphere の実行ファイルに専用アイコンを設定する。

## コンセプト
- モチーフ: 「球体 + ノード接続線」
- 意味付け:
  - 球体: 依存関係を俯瞰する 3D ビュー
  - 接続線とノード: コード要素間の依存リンク

## デザイン要素
- ベース形状: 円形グラデーションの球体
- 補助線: 球体上にクロスする接続線を 3 本配置
- ノード: 線の端点と交点に明るい点を配置
- 背景: 透過（Windows の各種表示面で馴染ませる）

## カラー
- 球体中心: `#4BA5FF`
- 球体外周: `#0D2558`
- 接続線: `#E1F2FF`
- ノード: `#F8FCFF`

## 出力仕様
- ファイル: `src/DepSphere.App/Assets/app-icon.ico`
- 形式: ICO（PNG 埋め込み）
- 主要サイズ: 256x256
- 透過: あり（32bit RGBA）

## 実装メモ
- `.ico` は PowerShell スクリプトで生成し、`ApplicationIcon` へ紐づける。
- 小サイズ表示で輪郭が潰れる場合は、ノード半径と線幅を再調整する。
